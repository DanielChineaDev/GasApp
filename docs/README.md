# Landing de GasApp

Esta carpeta contiene la web pública de GasApp, lista para servir con
**GitHub Pages**.

## Activar GitHub Pages

1. En GitHub → repo **GasApp** → *Settings* → *Pages*.
2. **Source:** *Deploy from a branch*.
3. **Branch:** `main` · **Folder:** `/docs` · Guardar.

En unos segundos quedará publicada en:
`https://danielchineadev.github.io/GasApp/`

(La landing es `index.html` y la política `privacy.html`.)

## Icono

La landing referencia `icon.png` (el logo de la app). Guarda el PNG (mismo
diseño que el de la app, idealmente ≥ 512×512) en esta misma carpeta como:

```
docs/icon.png
```

Y haz commit.

## URL de la política

Una vez Pages esté activo, usa esta URL en la ficha de Google Play:

```
https://danielchineadev.github.io/GasApp/privacy.html
```

(Recuerda rellenar antes los `[TU NOMBRE]` y `[TU EMAIL]` de la página.)
